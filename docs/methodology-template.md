## How to Use This Template

Every chapter has two methodology artifacts:

1. **`METHODOLOGY.md`** in the chapter directory -- the full technical document
2. **The in-page methodology section** -- a designed, readable version accessible from the chapter's navigation

Both are required. Both are held to the same standard. The in-page version is not a summary -- it is a designed presentation of the same content, written for a music journalist or historian with no ML background.

## Confidence Level Vocabulary

Every claim uses consistent language to signal certainty. These terms appear inline at the point of every claim they govern.

| Term | Meaning | When to use |
|------|---------|-------------|
| Documented | Appears in a primary source with specific citation | An interview transcript, a public statement, an official record |
| Verified | Cross-referenced across multiple independent sources | A statistic confirmed by Rock Hall records, MusicBrainz, and press |
| Reported | Appears in a single secondary source | A claim from one published article without independent confirmation |
| Estimated | Derived from incomplete data using documented assumptions | A criteria score built from partial data |
| Modeled | Output of an ML or statistical model with stated confidence bounds | A hazard ratio, a survival curve, a regression coefficient |
| Reconstructed | Built from fragmentary evidence using documented methodology | An influence citation assembled from multiple partial sources |
| Disputed | Subject to scholarly disagreement -- all major positions documented | Contested claims about the Hall's decision-making process |
| AI-generated | Produced by a generative AI model with human review | Docket briefs, artist summaries |
